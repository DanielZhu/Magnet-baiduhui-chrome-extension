### 功能与描述

- 百度惠首页精选频道商品浏览
- 查看精选商品和活动详情
- 查看详情页内评论
- 插件页面内设置
- 支持实时推送、音效、频率更改（可定制）
- 缓存最新的优惠列表（可定制）
- 支持VIP跳转，保障返利权利
